API Edi Covariance

Description

API Edi Covariance Platform memungkinkan Anda untuk mengakses data dan informasi terkait dengan platform kovariansi.

Istilah singkat :

  • device : merupakan sebuah perangkat untuk mengukur suhu edi covariance
  • logger : merupakan sebuah perangkat untuk mengirimkan data yang telah dikumpulkan dari device kedalam database

Base URL

The base URL for all API requests is:

http://localhost:3000

Endpoints

1. Autentikasi

GET /api

Deskripsi :

Mengembalikan pesan selamat datang setelah otentikasi berhasil.

Response

{
        "message": "Hello Dwi API, Authentication Is Now Complete Please Be Silent"
    }

This API uses the following error codes:

  • 401 Unauthorized: Authorization required.
  • 404 Not Found: Data tidak ditemukan.
